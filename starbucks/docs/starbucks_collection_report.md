# 스타벅스 매장 정보 수집 결과 보고서 (Starbucks Store Data Collection Report)

본 보고서는 스타벅스 코리아 전국 매장 정보 수집 파이프라인의 실행 결과와 수집된 데이터의 통계를 요약하여 제공합니다.

---

## 1. 수집 개요
- **수집 완료 시점**: 2026년 7월 10일
- **수집 엔진**: Python (`requests`, `pandas`)
- **적용 로직**: 
  - 17개 시도 코드를 이용한 동적 순회 수집
  - 서버 과부하를 방지하기 위한 요청 간 랜덤 딜레이 (`0.3 ~ 0.8초`)
  - 네트워크 및 서버 일시 오류 처리를 위한 지수 백오프 기반 재시도 (최대 5회)
- **저장 위치**: `starbucks/data/starbucks_stores.csv` (인코딩: `utf-8-sig`)

---

## 3. 크롤러 구동 가이드

본 수집 파이프라인은 워크스페이스 내 공통 `.venv`를 사용하여 다음과 같이 실행합니다.

### A. 의존성 패키지 설치
공통 가상환경에 데이터 처리와 HTTP 통신을 위해 필요한 라이브러리(`pandas`, `requests`)가 설치되어 있어야 합니다. (이미 설치 완료됨)
```bash
uv pip install requests pandas
```

### B. 수집기 실행
워크스페이스 루트 디렉터리에서 다음 명령어를 실행합니다.
```bash
uv run python starbucks/src/collector.py
```

### C. 데이터 결과물 확인
수집 작업이 끝나면 `starbucks/data/starbucks_stores.csv` 경로에 엑셀과 호환되는 UTF-8-BOM 형식으로 데이터가 적재됩니다.
정상적으로 데이터가 적재되었는지 다음과 같이 확인할 수 있습니다.
```python
import pandas as pd
df = pd.read_csv('starbucks/data/starbucks_stores.csv')
print(df.head())
```
